# Sweet Haven Bakery Website

A modern, responsive website for a bakery business showcasing products and services.

## Features

✨ **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices

🎨 **Modern UI** - Beautiful gradient backgrounds and smooth animations

📱 **Mobile Optimized** - Touch-friendly interface for all devices

🛒 **Product Showcase** - Display bakery products with prices and descriptions

📞 **Contact Form** - Easy way for customers to get in touch

✅ **Smooth Scrolling** - Professional navigation experience

🎯 **Call-to-Action** - Strategic buttons to encourage customer engagement

## Sections

1. **Navigation Bar** - Sticky header with links to all sections
2. **Hero Section** - Eye-catching banner with main message
3. **Products** - Grid showcase of bakery items
4. **About Us** - Story and features of the bakery
5. **Contact** - Business information and contact form
6. **Footer** - Social links and copyright

## Products Included

- Croissants

- Sourdough Bread
- Cupcakes
- Pastries
- Custom Cakes

## Customization

### Colors
Edit the color variables in `styles.css`:
```css
--primary-color: #D2691E;
--secondary-color: #8B4513;
--accent-color: #FFD700;
```

### Business Information
Update the contact section in `index.html` with your:
- Business name
- Address
- Phone number
- Email
- Operating hours

### Products
Modify the products grid in `index.html` to add/remove/change products and prices.

## Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No dependencies or build tools required!
